# PixelSwift Frontend
This is the React-based user interface for the PixelSwift Image Converter.

### Tech Stack
* **React** (TypeScript)
* **CSS3** (Glassmorphism & Responsive Design)

### Features
* **Drag & Drop**: Interactive upload zone for images.
* **Live Preview**: View your image before conversion.
* **Theme Toggle**: Support for Light and Dark modes.
