# PixelSwift Backend
The high-performance image processing engine powered by Node.js and Sharp.

### Tech Stack
* **Node.js & Express**
* **Sharp Engine**: For high-fidelity image manipulation.
* **Multer**: For secure file handling.

### Specialized Logic
* **PSD Support**: Automatically converts PSD files to high-quality TIFF for Photoshop compatibility.
* **SVG Support**: Handles vector-to-raster conversion with precision.
